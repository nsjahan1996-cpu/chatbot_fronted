# Chatbot Frontend

A simple interactive chatbot built using **HTML, CSS, and JavaScript**.  
This project features a floating chat icon, hover options for Telegram/WhatsApp/Messenger, and a chat window where users can send messages and get simple auto-replies.


## Features

- Floating **chat icon** on the bottom right
- **Hover icons** for Telegram, WhatsApp, Messenger
- Smooth slide-in **chatbot window**
- **User can type messages**
- **Simple keyword-based auto-reply**
- Responsive layout for desktop and mobile


## How to Use

1. Clone or download the repository
2. Open `index.html` in your browser
3. Click the chat icon to open the chatbot
4. Type your message in the input box and press **Send** or **Enter**
5. Chatbot will reply automatically based on simple keywords
